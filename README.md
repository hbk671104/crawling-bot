# crawling-bot

A crypto crawler specializing in code frequency of repos.

## Guide

### 0. Create .env file

```sh
touch .env
```

### 1. Fill up environmental variables

-   GITHUB_PERSONAL_TOKEN
-   LEANCLOUD_APP_ID
-   LEANCLOUD_APP_SECRET
-   LEANCLOUD_SERVER_URL

### 2. Run!

```sh
node app.js
```
